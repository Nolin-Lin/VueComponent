# vue-component

> 用vue开发的一些小组件

## 技术栈

vue2 + vue-loader + webpack + sass + ES6

## 开发

``` bash
git clone https://github.com/Nolin-Lin/VueComponent.git  

cd VueComponent

npm install(国内推荐cnpm)  或 yarn(推荐)

npm run dev
```
